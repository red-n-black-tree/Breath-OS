Freehand Tool

  • Draw irregular hand-drawn Curve entities or 3D Polyline entities.

Tool Operation

  1. Click and hold at starting point of curve.
  2. Drag the cursor to draw.
  3. Release mouse button to stop.
  4. (optional) End curve at starting point to draw closed shape.
  5. Esc = Cancel operation.

Modifier Keys

  • Shift = Draw 3D Polyline entity

Advanced Operations

  • Drawing 3D Polyline Entities
