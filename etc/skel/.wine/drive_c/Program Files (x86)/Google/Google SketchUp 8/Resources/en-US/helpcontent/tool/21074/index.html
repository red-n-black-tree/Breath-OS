Paint Bucket Tool

  • Assign materials and colors to entities.

Tool Operation

  1. Select a materials library using drop down list in Materials Browser.
  2. Select a material from materials library.
  3. Click on faces to paint.

Modifier Keys

  • Shift = Paint all faces connected to face that you click on regardless of context
  • Ctrl = Paint all faces connected to face that you click on within current context
  • Shift+Ctrl = Adjacent replace
  • Alt = Sample material for painting

Advanced Operations

  • Face Painting Rules
  • Fill Options
  • Sampling a Material
