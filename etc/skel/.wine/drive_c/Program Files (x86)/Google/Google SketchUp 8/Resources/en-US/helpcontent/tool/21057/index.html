Protractor Tool

  • Measure angles and create angled Construction Line entities.

Tool Operation

  1. Place protractor’s center at vertex (where two lines meet) of angle.
  2. Click to set vertex.
  3. Move cursor in circle until touching start of angle (one of the lines).
  4. Click to set start of angle.
  5. Move cursor in circle until touching end of angle (other line).
  6. Click to measure angle.
  7. Esc = Cancel operation.

Modifier Keys

  • Ctrl = Toggles create construction line
  • Shift = Lock inference

Advanced Operations

  • Locking the Protractor Tool to its Current Orientation
  • Creating Precise Angles
