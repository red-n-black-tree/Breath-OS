3D Text Tool

  • Create 3 dimensional text using any font.

Tool Operation

  1. Type text in text box. 
  2. Press Fonts... button and select font, font style, points and height.
  3. Check Filled checkbox.
  4. Check Extruded checkbox.
  5. Press Place button.
  6. Move cursor to position text.
  7. Click to finish position operation.

Modifier Keys

  • None
