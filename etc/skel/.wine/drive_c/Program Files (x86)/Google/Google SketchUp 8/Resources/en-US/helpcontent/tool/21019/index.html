Eraser Tool

  • Erase entities.

Tool Operation

  • Click on entity to erase. Alternatively, hold down mouse button drag over entities. All entities are erased when mouse button is released.

Modifier Keys

  • Shift = Hide entities
  • Ctrl = Soften and smooth entities
  • Shift+Ctrl = Unsoften and unsmooth entities

Advanced Operations

  • Hiding Lines
  • Softening/Unsoftening Edges
