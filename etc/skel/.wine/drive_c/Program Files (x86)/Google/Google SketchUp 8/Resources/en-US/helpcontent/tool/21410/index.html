Dimension Tool

  • Place Dimension entities.

Tool Operation

  1. Click at starting point of dimension.
  2. Move cursor.
  3. Click at ending point of dimension.
  4. Move cursor perpendicular to dimension.
  5. Click to fix position of dimension string.
  6. Esc = Cancel operation.

Modifier Keys

  • None.

Advanced Operations

  • Placing Radius Dimensions
  • Placing Diameter Dimensions
  • Toggling Dimension Type
