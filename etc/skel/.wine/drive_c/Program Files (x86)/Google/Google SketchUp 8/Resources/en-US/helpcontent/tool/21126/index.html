Axes Tool

  • Move or reorient drawing axes.

Tool Operation

  1. Move cursor to point in drawing area for new origin.
  2. Click to establish origin.
  3. Move cursor away from origin to set direction for red axis.
  4. Click to accept direction.
  5. Move cursor away from origin to set direction for green axis.
  6. Click to accept direction.
  7. Esc = Cancel operation.

Modifier Keys

  • None.

Advanced Operations

  • Resetting the Drawing Axes
