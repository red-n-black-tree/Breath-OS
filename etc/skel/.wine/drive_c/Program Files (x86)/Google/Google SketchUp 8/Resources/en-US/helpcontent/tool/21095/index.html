Polygon Tool

  • Draw Polygon entities.

Tool Operation

  1. Click to place center point.
  2. Move cursor away from center point to define radius.
  3. Click to finish polygon.
  4. Esc = Cancel operation.

Modifier Keys

  • Shift (before first click) = Lock polygon to its current orientation
